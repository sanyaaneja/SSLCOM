Accessibility Statement

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Staff-Student Liaison Committee is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Technical Specifications

Accessibility of Staff-Student Liaison Committee relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

These technologies are relied upon for conformance with the accessibility standards used.

Accessibility Choices

The steps I undertook to make this website as accessible as possible include:

  1. Adding alt to images: this helps the user identify what the image is if theyâ€™re using a screen reader or the image is unable to load.
  2. Color Contrast: By making the background a contrasting shade than the text above, I created a distinction, making the content readable.
  3. Consistent Navigation: By using the same navigation menu on each webpage, I made it easier for the user to traverse the website.
  4. Navigation of website at 200%: Since several users might want to zoom into the screen to have a clearer picture, the website can function and is easy to navigate when zoomed up to 200%
