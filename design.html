Design Reasoning

The website consists of two layouts: desktop and mobile. The desktop layout is developed to fit a width of 992px and above. Since the website is mobile-first responsive, the width for the mobile layout isn't defined.

Menu System

The navigation bar serves as the menu. In the desktop version, it consists of a row with different links at each interval. In the mobile layout, the various linked web pages are present as several rows distinguished by a border.

The menu differs for the two layouts since it is hard to access the menu on the mobile if it is present as a single row. Hence, in order to make it accessible for the user, it is divided into multiple rows.
